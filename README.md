QSP model of Atopic Dermatitis, https://pubmed.ncbi.nlm.nih.gov/33894014/

The repository contains the matlab model file for the QSP model of AD based on the original python model at https://github.com/Tanaka-Group/AD_QSP_model

The file to run the default simulation in matlab is [convertedMatlab/runADmodel.m](https://github.com/jaimit19/qsp-ad/blob/main/convertedMatlab/runADmodel.m)
